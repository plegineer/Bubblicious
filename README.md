# Bubblicious
## クローンから実行までの手順
1. ```$ git clone -b develop https://github.com/plegineer/Bubblicious.git```
2. ```$ pod install ```(ただしCocoapodsがインストールされてない場合は、https://cocoapods.org/ からインストール)
3. ```$ open Bubblicious.xcworkspace```
4. Xcodeの▶ボタンを押して実行
